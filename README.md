hyperfuzz
=========

A streamlined symbolic execution / fuzzer based on valgrind, stp, and SmartFuzz.
Work done for masters thesis in Cyber Security (NYU Polytechinc School of Engineering 2015).

Package Dependencies:
====================
	easygit
	build-essential
	autoconf
	automake
	autotools-dev
	m4
